## Descrição

Essa aplicação foi construída com React utilizando motion para animações. É uma maneira melhor de expor meus projetos do que apenas deixá-los no github.

## Instruções de intalação

Antes de começar, certifique-se de ter instalado:
- Node.js
- npm ( instalado junto com Node.js )

### Clonando o projeto
``` bash
git clone LINK_DO_REPOSITÓRIO
cd portf-lio
```

### Instalando dependências
```bash
npm install
```

### Rodando o projeto
```bash
npm run dev
```
Após iniciar, o terminal vai mostrar o link do site.
